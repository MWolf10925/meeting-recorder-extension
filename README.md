# Meeting Recorder Extension

A Chrome extension and dashboard concept for recording online meetings, generating transcripts, and storing summaries.

## Overview

This project was built as a practical web and extension development prototype. The goal was to detect meeting tabs, record audio, process transcripts, generate summaries, and save meeting history to a dashboard.

## Features

- Chrome extension popup
- Meeting tab detection
- Start/stop recording concept
- Web dashboard
- Meeting list and detail pages
- Transcript and summary storage

## Tech Used

- Chrome Extension Manifest V3
- JavaScript / TypeScript
- Next.js
- Supabase
- Web dashboard UI

## Status

Prototype / internship-style technical project.
